# rv-presentation-bot
Интерактивный Telegram-бот-презентация разработчика. Проект демонстрирует навыки Python-разработки через реальный рабочий продукт: Telegram-бот с поддержкой FastAPI, aiogram, SQLAlchemy async и Alembic. Бот служит «цифровым двойником» автора и используется как живое портфолио для HR и технических специалистов.
